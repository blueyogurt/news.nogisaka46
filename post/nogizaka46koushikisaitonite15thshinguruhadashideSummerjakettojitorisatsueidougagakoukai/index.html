【タイトル＆ジャケット写真が決定！！】乃木坂46 15thシングル！！
2016.07.01
2016/07/27（水）に発売する乃木坂46の15thシングルのタイトルが「裸足でSummer」、さらにジャケット写真も決定しました！！
６月上旬に千葉県九十九里にあるスタジオで撮影された今回のジャケット写真のテーマは、ズバリ"自撮り"。
世界的にムーブメントとなっている「自撮り / セルフィー」をコンセプトに、メンバー同士でしか撮れない自然な表情や、普段は見ることができない素の姿など、"自撮り棒"を使って自由に撮影。
メンバーの自然体で楽しい姿が楽しめるジャケット写真となっています！！
梅雨の時期という事もあり、撮影当日の朝は大雨でしたが、撮影が始まると運良く晴れて、奇跡的に夏らしい撮影日となりました！！
今後も15thシングル「裸足でSummer」の様々な情報をお届けしていきますので、是非チェック！！
●Type-A / 撮影者：齋藤飛鳥
15thシングルでセンターを務める齋藤飛鳥ソロの自撮り。普段見れない無防備な表情がジャケット写真となっています。

●Type-B / 撮影者：橋本奈々未
ちょうど4人の目の前に大きなシャボン玉が来た驚きの表情がジャケット写真となっています。

●Type-C / 撮影者：桜井玲香
色々と工夫をして撮影していたグループで、鏡を使って遊んでいる1枚がジャケット写真となっています。

●Type-D / 撮影者：中元日芽香
自撮り棒を上に向けて、サークル状に並んだ写真がジャケット写真となっています。

●通常盤 / 撮影者：齋藤飛鳥
齋藤飛鳥だけ１人前に来て、選抜全員を自撮りしている、今回のジャケットを象徴するような写真です。
